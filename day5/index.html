Type of Selectors

  • element selector
  • id selector
  • class selector
  • universal selector
  • grouping selector

The CS Element Selector

ALL HTML TAGS BASED ON THE ELEMNT NAME IS CALLED AS ELEMENT SELECTOR 

        examples:p,h1,h3,img,b,i etc....,

        h1{
            text_align:center;
            color:blue;
        }

The CS Id Selector

the id selector uses the id attribute of an html element to select a specific element. to select an element with the hash(#) character, followed by the id element 

        examples: HTML element with id="para1"

        #para1{
            text_align:center;
            color:blue;
        }

The CS Class Selector

the class selector selects html elements with a specific class attribute. to select elements with a specific class, write a period(.) character, followed by the class name

        examples: HTML element with class="intro"
        .intro{
            text_align:center;
            color:blue;
        }

The CS Universal Selector

the universal selector (*) selects all elements on the page

        *{
            text_align:center;
            color:blue;
        }

The CS Grouping Selector

the grouping selector selects all the elements with the same style definitions. to group selectors, separate each selector with a comma

        h1,h2,p{
            text_align:center;
            color:blue;
        }
